Multiplication Calculator

Write a program to accept two numbers from the user and calculate their multiplication. 

# Prompt the user to input the first number and convert it to an integer, assigning it to the variable 'num1'
num1 = int(input("Enter first number: "))

# Prompt the user to input the second number and convert it to an integer, assigning it to the variable 'num2'
num2 = int(input("Enter second number: "))

# Calculate the multiplication of the two numbers and assign the result to the variable 'res'
res = num1 * num2

# Display the result of the multiplication
print("Multiplication is", res)
