Accept List of Float Numbers

Use a loop to accept 5 float numbers from the user and store them in a list. 

numbers = []

# Specify the list size
list_size = 5

# Loop to accept 5 float numbers from the user
for i in range(0, list_size):
    # Prompt the user to enter a float number at the current location
    print("Enter number at location", i, ":")
    # Accept a float number from the user and add it to the list
    item = float(input())
    numbers.append(item)

# Display the list of float numbers entered by the user
print("User List:", numbers)
