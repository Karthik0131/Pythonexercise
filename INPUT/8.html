Format Variables using string.format()

This script uses the string.format() method to format the following three variables: totalMoney, quantity, and price. 

# Given variables
totalMoney = 1000
quantity = 3
price = 450

# Create a formatted string using string.format() method
statement = "I have {} dollars so I can buy {} football for {:.2f} dollars."

# Format the string with the variables
formatted_statement = statement.format(totalMoney, quantity, price)

# Print the formatted statement
print(formatted_statement)
