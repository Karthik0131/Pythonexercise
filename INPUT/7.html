Accept Three Strings

This script takes three names as input from a user in a single input call. 

# Prompt the user to input three strings in a single input call and split the input into three variables
str1, str2, str3 = input("Enter three strings separated by spaces: ").split()

# Display the entered strings
print('Name1:', str1)
print('Name2:', str2)
print('Name3:', str3)
