Reverse a List in Python

Given: list1 = [100, 200, 300, 400, 500]

Expected output: [500, 400, 300, 200, 100]

# Solution 1: list function reverse()
list1 = [100, 200, 300, 400, 500]
list1.reverse()
print(list1)

# Solution 2: Using negative slicing
list1 = [100, 200, 300, 400, 500]
list1 = list1[::-1]
print(list1)
