Remove Empty Strings from the List of Strings

Given: list1 = ["Mike", "", "Emma", "Kelly", "", "Brad"]

Expected output: ["Mike", "Emma", "Kelly", "Brad"]

list1 = ["Mike", "", "Emma", "Kelly", "", "Brad"]

# Solution: List comprehension to remove empty strings
res = [string for string in list1 if string]
print(res)
